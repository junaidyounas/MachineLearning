# Artificial Intelligence — Machine Learning & Deep Learning

**Government of Pakistan · NAVTTC · Prime Minister's Hunarmand Pakistan Program**  
*"Skills for All"*

This repository contains coursework, notebooks, and assignments for the **Artificial Intelligence (Machine Learning & Deep Learning)** vocational training program.

| | |
|---|---|
| **Duration** | 3 Months |
| **Class Hours** | 4 hours/day |
| **Theory : Practical** | 20% : 80% |
| **Batch Size** | 25 students |
| **Learning Place** | Classroom / Lab |

---

## Course Objective

Develop employable skills and hands-on practice in Artificial Intelligence, with specialization in:

- **Natural Language Processing (NLP)**
- **Microsoft Azure AI Associate**

The program is participant-centered and practically oriented. By the end of the course, trainees should be able to work professionally in AI/ML environments and apply modern techniques to real-world problems.

---

## Learning Outcomes

After completing this course, you will:

- Understand core concepts of **Artificial Intelligence** and **Machine Learning**
- Apply state-of-the-art machine learning techniques
- Perform **Exploratory Data Analysis (EDA)**
- Design, evaluate, and deploy ML/DL models
- Use industry-standard libraries: **scikit-learn**, **pandas**, **numpy**, **TensorFlow**, **PyTorch**, and **Keras**

### NLP Specialization

- Build TensorFlow pipelines for NLP
- Extract information from textual data
- Apply text normalization, stemming, and lemmatization
- Use evaluation metrics for NLP models
- Build POS tagging and Named Entity Recognition (NER) models
- Work with word embeddings and deep learning techniques
- Deploy NLP models

### Microsoft Azure AI

- Use Microsoft Azure APIs for Machine and Deep Learning on numerical, image, and text data

## 12-Week Course Schedule

### Week 1 — Introduction, Linux & Python Fundamentals

| Day | Topics |
|-----|--------|
| Day 1 | Introduction to AI, course overview, job market, software installation (Anaconda, VSCode, PyCharm) |
| Day 2 | Debian basics: `pwd`, `cd`, `ls`, `cat`, `sudo`, `mkdir`, `rm`, `cp`, `mv`, `chmod`, `ps`, `kill`, environment variables |
| Day 3 | Python: values, expressions, numbers, booleans, strings, operators, variables, input & type casting |
| Day 4 | Data structures: lists, tuples, dictionaries, sets |
| Day 5 | Conditionals (`if`/`elif`/`else`), `break`/`continue`/`pass`, loops, list comprehension, iterators |

### Week 2 — Python, OOP & Descriptive Statistics

| Day | Topics |
|-----|--------|
| Day 1 | Functions, scope, lambda, `map`/`filter`, file handling, exception handling |
| Day 2 | OOP: classes, objects, inheritance, access specifiers, association/aggregation/composition |
| Day 3 | Polymorphism, operator overloading, magic methods, abstract classes, data classes |
| Day 4 | Data types (structured/unstructured), quantitative & qualitative variables |
| Day 5 | Central tendency (mean, mode, median), dispersion (variance, std dev), position (Z-score, percentile) |

### Week 3 — Statistics, Probability & NumPy

| Day | Topics |
|-----|--------|
| Day 1 | Correlation, univariate/bivariate/multivariate plots, probability |
| Day 2 | Joint/marginal/conditional probability, distributions, Bayesian probability |
| Day 3 | NumPy: array creation, attributes, reshape, operations |
| Day 4 | Indexing, broadcasting, type casting, arithmetic & universal functions |
| Day 5 | Pandas: Series, DataFrame, selection, operations, missing data |

### Week 4 — Pandas, Seaborn & ML Introduction

| Day | Topics |
|-----|--------|
| Day 1 | Merging/joining, GroupBy, binning, Pandas plotting |
| Day 2 | Seaborn: distribution, categorical, and matrix plots |
| Day 3 | ML introduction, classical ML pipeline |
| Day 4 | Supervised learning, regression & classification, cost function, optimizer |
| Day 5 | Univariate linear regression with gradient descent (with & without vectorization) |

### Week 5 — Machine Learning I

| Day | Topics |
|-----|--------|
| Day 1 | Multivariate linear regression |
| Day 2 | Polynomial regression |
| Day 3 | Logistic regression (binary classification) |
| Day 4 | Logistic regression (multiclass classification) |
| Day 5 | Code practice |

### Week 6 — NLP & Machine Learning II

| Day | Topics |
|-----|--------|
| Day 1 | NLP introduction, syntax/semantics/pragmatics, NLTK/SpaCy, noise removal |
| Day 2 | Tokenization, stemming, lemmatization, POS/NER tagging, BoW, document similarity |
| Day 3 | Model testing, evaluation metrics, dataset imbalance & augmentation |
| Day 4 | Support Vector Machine (SVM), Decision Tree |
| Day 5 | Random Forest (bagging), Build Your CV, **Mid-term Exam** |

### Week 7 — Deep Learning I

| Day | Topics |
|-----|--------|
| Day 1 | Boosting |
| Day 2 | MLP, forward/backward pass, activation functions, backpropagation, regularization |
| Day 3 | Neural networks with TensorFlow / PyTorch / Keras |
| Day 4 | CNN: 2D (image) and 1D (text) classification |
| Day 5 | Neural network code practice |

### Week 8 — Deep Learning II (RNNs)

| Day | Topics |
|-----|--------|
| Day 1 | Recurrent Neural Networks (RNNs) |
| Day 2 | Long Short-Term Memory (LSTM) |
| Day 3 | LSTM code practice |
| Day 4 | Gated Recurrent Unit (GRU) |
| Day 5 | GRU code practice |

### Week 9 — Deep Learning II (Embeddings & Sequence Models)

| Day | Topics |
|-----|--------|
| Day 1 | Word embeddings: Word2Vec, CBOW, Skip-gram |
| Day 2 | Gensim and custom embedding training |
| Day 3–5 | Sequence models: 1-to-1, 1-to-many, many-to-1, many-to-many |

### Weeks 10–12 — Project, Azure AI & Assessment

| Week | Topics |
|------|--------|
| Week 10 | Bi-directional LSTM/RNN, attention mechanism, **employable project** (2 weeks) or on-job training |
| Week 11 | Microsoft Azure AI: service selection, security, resource management, anomaly detection, image/video processing |
| Week 12 | Azure NLP, knowledge mining, conversational AI solutions |

## Work Ethics

The course emphasizes ten essential workplace ethics:

1. **Attendance** — Be punctual and plan absences responsibly
2. **Character** — Honesty, completing tasks correctly and promptly
3. **Teamwork** — Collaborate and support peers
4. **Appearance** — Professional presentation and hygiene
5. **Attitude** — Positive, responsible, safety-conscious
6. **Productivity** — Quality work, cooperation, ownership
7. **Organizational Skills** — Time management and continuous improvement
8. **Communication** — Clear written and verbal communication
9. **Cooperation** — Follow rules, adapt to change
10. **Respect** — Value diversity and others' perspectives
